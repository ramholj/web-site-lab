# web-site-lab

## 项目简介

`web-site-lab` 是一个用于归档和发布多个独立 HTML 页面的仓库。  
这里存放的页面主要用于测试、原型展示、学习记录或临时用途，不针对任何特定的域名或具体网站。

每个 HTML 文件均为独立页面，可直接在浏览器中打开运行，或通过 GitHub Pages 等服务进行访问。

## 目录说明

```
web-site-lab/
├── index.html          # 仓库首页 / 页面导航
├── page-1.html         # 示例页面一
├── page-2.html         # 示例页面二
└── assets/             # 共用资源目录（可选）
    ├── css/
    ├── js/
    └── images/
```

- 根目录下的 `.html` 文件为各个独立页面。
- `assets/` 目录用于存放页面可能用到的公共样式、脚本或图片。
- 部分页面可能包含子目录，用于组织相关的资源文件。

## 页面归档说明

本仓库中的页面来源多样，可能包括：

- 个人学习或实验过程中的产出
- 临时测试页面的快照
- 原型设计或概念验证文件
- 其他无需独立维护的静态页面

每个页面通常都自包含（或引用 `assets/` 下的资源），可直接使用。  
页面内容不保证持续更新或适配所有浏览器环境。

## 维护说明

- 本仓库不定期更新，新增或删除页面不另行通知。
- 欢迎提交 Issue 或 Pull Request 来修正错误、改进页面或增加新内容。
- 请勿在本仓库中存放涉及敏感信息、商业机密或侵犯他人权益的内容。
- 所有页面均按“原样”提供，不提供任何形式的保证或技术支持。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 直接在浏览器中打开任意 `.html` 文件即可查看页面效果。
3. 如需部署到 GitHub Pages，请确保仓库设置中已启用 Pages 功能，并选择正确的分支与目录。
